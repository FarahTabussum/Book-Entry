# 📚 Lumina – Personal Library

Lumina is a modern single-page personal library web application built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript (ES6)**. It allows users to organize their reading journey by adding books, managing reading tasks, and viewing recently added books through a clean and responsive interface.

## Live Demo

[Live Link](https://lumina-library-farah.surge.sh)

## 🚀 Features

- 📖 Personal library dashboard
- ➕ Add new books with:
  - Book title
  - Author
  - Genre (custom dropdown)
  - First date
  - Total pages
- 📚 Sidebar navigation
- 🔍 Search bar
- 📌 Current reading section with:
  - Reading progress
  - Progress bar
  - Current page count
- ✅ Reading task manager
  - Add new tasks
  - Mark tasks as completed
  - Completed tasks display with strikethrough
- 📕 Recently Added section
- ⚡ Quick Add card
- 🎨 Modern UI built with Tailwind CSS
